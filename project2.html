Web Page Blog for Human Computer Interaction

Project Description

This project was developed for the Human Computer Interaction course. NutriWell is a wellness blog website designed to help users improve their lifestyle through nutrition and health information. The website features a clean and intuitive navigation menu, visually appealing design with food imagery, and multiple sections including home, about, recipes, contact, and a meal tracker tool. The blog provides users with resources and guidance for maintaining a healthy lifestyle.

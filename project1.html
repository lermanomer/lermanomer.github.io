GUI Builder for Enterprise Computing

Project Description

This project was developed for the Enterprise Computing class. The GUI Builder is a shopping cart application that allows users to manage items in their cart. Users can enter item IDs, quantities, and additional details for each item. The application displays a running subtotal and provides various controls to search for items, add items to the cart, delete items, and proceed to checkout.
